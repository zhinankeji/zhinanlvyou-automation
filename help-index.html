安装指南

从下载到使用，全程图文教程。遇到问题参考 FAQ 或联系微信 hibeike。

一、下载并解压

1 点击下载
在首页点击绿色“免费下载”按钮，获取 ZhinanAI_Setup_v1.0_offline.zip (311MB)。
2 解压到桌面
将 ZIP 文件解压到桌面或任意文件夹，目录结构如下：
ZhinanAI-DeepSeek-Offline/
├── install.ps1 ← 安装脚本
├── uninstall.ps1
├── packages/ ← 离线安装包
└── README.md

二、运行安装

3 右键 install.ps1 → “用 PowerShell 运行”
找到 install.ps1，右键选择“用 PowerShell 运行”。
⚠ 如果没有这个选项，打开 PowerShell 输入：
PS> Set-ExecutionPolicy RemoteSigned -Scope CurrentUser -Force
PS> .\install.ps1
4 等待自动安装
系统自动安装 Python → pip → Hermes Agent → Node.js → Git → ripgrep → FFmpeg，无需人工干预。
==========================================
指南帮 AI 助手 - 安装程序 v1.0
==========================================

--- 步骤 1/12: 安装 Python 3.11 ---
[信息] 正在复制 Python 3.11.9 ...
[成功] Python 3.11.9 已安装
--- 步骤 2/12: 安装 pip ---
[成功] pip 安装成功
--- 步骤 3/12: 安装 Hermes Agent ---
[成功] Hermes Agent 安装完成
...
[成功] PATH 环境变量配置完成

三、配置 DeepSeek API Key

5 输入 API Key
安装到第 9 步时会自动弹出提示，请输入您的 DeepSeek API Key：
--- 步骤 9/12: 配置 DeepSeek API ---
请输入您的 DeepSeek API Key：
(如果没有，访问 https://platform.deepseek.com/ 注册获取)
API Key: sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

[成功] 配置文件已创建
ℹ 如果没有 Key，可以先按回车跳过，安装完成后编辑 %APPDATA%\ZhinanAI\config.yaml 填入。

四、安装完成

6 安装完成
桌面会多出“指南帮 AI 助手”快捷方式：
==========================================
安装完成！
==========================================

已安装的组件：
- Python 3.11.9 / Node.js 22.14.0 / Git 2.48.1 / ripgrep 14.1.1 / FFmpeg / Hermes 0.17.0

配置文件: %APPDATA%\ZhinanAI\config.yaml
桌面快捷方式: 指南帮 AI 助手.lnk
7 双击桌面快捷方式启动
双击桌面“指南帮 AI 助手”，启动 Hermes Agent。
⚠ 第一次启动时输入 /model ，选择 provider=deepseek, model=deepseek-v4-pro

五、开始使用

就绪！可以开始使用了
在 Hermes 终端中输入任务让 AI 处理：
> 帮我分析这个目录下的代码
正在分析 ...

> 写一个 Python 脚本处理 Excel 文件
正在生成 ...

附录：常见问题

问题：右键没有“用 PowerShell 运行”
打开 PowerShell，输入：
PS> Set-ExecutionPolicy RemoteSigned -Scope CurrentUser -Force
PS> .\install.ps1
问题：找不到 hermes 命令
关闭终端重新打开，或注销重新登录 Windows。
问题：如何卸载
开始菜单 → 指南帮 AI 助手 → 卸载，或双击 uninstall.ps1。
