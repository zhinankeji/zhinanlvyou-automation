常见问题

❓ 需要联网吗？
是的，使用本产品需要保持联网。所有 AI 对话和任务处理都通过 DeepSeek API 在线完成。不支持纯离线运行（除非自行配置本地模型）。但 安装过程本身不需要联网，离线包已内置全部依赖组件。
❓ 需要管理员权限吗？
不需要。安装到用户目录 %LOCALAPPDATA%\\ZhinanAI，环境变量通过 setx 配置用户级 PATH。
❓ 和 ChatBox / Open WebUI 有什么区别？
指南帮 AI 助手是基于 Hermes Agent 的 编程/任务 AI 助手，不是聊天界面。它可以直接执行代码、搜索文件、操控终端，适合开发者和 IT 运维使用。
❓ 能换其他大模型吗？
本版本仅预配置 DeepSeek。如需其他模型（如通义千问、智谱 GLM），需自行获取对应 API Key 后通过 /model 命令切换。各模型的联网要求不变。
❓ 怎么卸载？
开始菜单 → 指南帮 AI 助手 → 卸载，或直接运行 uninstall.ps1。
❓ 企业批量部署怎么操作？
企业版用户提供批量部署脚本，GPO 下发或 SCCM 推送均可。联系微信 hibeike 获取。
